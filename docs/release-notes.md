# DCJ Free PDF Mailer リリースノート

## 1. リリース概要

このドキュメントは、DCJ Free PDF Mailerの販売・納品・保守に向けた最終整理として、現時点の完成済み機能、本番確認済み内容、ファイル構成、運用上の注意、今後の改善候補をまとめるものです。

本版は、無料PDF配布フォームの作成、PDFリンク付きメール送信、送信ログ管理、お知らせ受信同意、購読者リスト管理、配信用CSV出力、配信停止URL、reCAPTCHA v3によるスパム対策までを一通り扱える状態です。

導入先のWordPress環境、サーバー環境、メール送信環境、運用ルールによって、追加確認や調整が必要になる場合があります。

### v1.3.0 リリース項目

v1.3.0では、管理画面説明文の整理、購読者の手動追加、購読者CSVインポートを追加しました。

- 管理画面説明文まとめ改善
- 購読者の手動追加を実装
- 管理画面の購読者リストから、同意確認済みのメールアドレスを1件ずつ追加可能
- 入力項目はメールアドレス、言語、登録元メモ、同意確認済みチェック
- 手動追加の言語は `ja` または `en` を必須にし、言語未指定の新規登録は許可しない
- 既に登録済み、または配信停止済みのメールアドレスは追加不可
- 配信停止済みアドレスを自動で購読中に戻さない
- 手動追加した購読者は、既存の購読者リスト表示とCSV出力に同じ形式で反映
- 手動追加フォーム付近に成功・エラー表示を追加
- 購読者CSVインポート仕様メモを作成
- 購読者CSVインポートを実装
- CSVファイル選択、2つの確認チェック、プレビュー、インポート実行に対応
- CSV形式はヘッダー行あり、必須列 `email` / `lang`、任意列 `source_note` / `consent_confirmed`
- CSVインポートの `lang` は `ja` または `en` を必須にし、言語未指定の新規登録は許可しない
- `consent_confirmed` は `yes` / `1` / `true` を同意済みとして扱う
- プレビューで読み込み件数、登録予定件数、スキップ件数、エラー一覧を確認可能
- CSVインポート実行で登録成功件数、スキップ件数、エラー件数を表示
- 既に登録済み、配信停止済み、CSV内重複、不正メール、言語未指定、言語不正、同意未確認の行はスキップ
- CSVインポートした購読者は、既存の購読者リスト表示とCSV出力に同じ形式で反映
- DB構造、既存CSV出力処理、配信停止URL処理は変更なし
- 既存データの言語未指定は後方互換として残しつつ、手動追加とCSVインポートでは新しい言語未指定データを増やさない

## 6. 本番運用時の注意

- PDF URLには、ブラウザで直接開けるURLを設定してください。
- `/wp-content/uploads/dlm_uploads/` 配下など、直接アクセスが制限されるURLは避けてください。
- メール到達は、サーバー環境、DNS設定、SMTP設定、受信者側のメール環境に依存します。
- メール送信診断は簡易チェックです。メール到達を保証するものではありません。
- Gmailなどでは、サーバー経由表示や迷惑メール判定が出る場合があります。
- メールアドレス・CSVは個人情報として慎重に扱ってください。
- お知らせ配信、販売案内、クーポン案内には、購読者リストの「購読中」だけを使ってください。
- 配信停止の人には送らないでください。
- 送信ログCSVは履歴確認用です。配信用メールアドレス一覧としては使わないでください。
- 購読者リストCSVは、購読者の管理・バックアップ用として使ってください。
- 配信用CSVは、メール配信サービスへの手動インポート補助用です。全言語・日本語のみ・英語のみで出力できます。インポート前にCSV内容、同意状況、配信停止者の扱い、配信サービス側の同意管理・配信停止管理を確認してください。
- 既存PDF設定で `{{newsletter_unsubscribe_block}}` を使う場合は、メール本文へ手動追加してください。
- reCAPTCHA v3を使う場合は、本番ドメインで取得したSite Key / Secret Keyを使ってください。
- LocalWPなど登録ドメインと異なる環境では、reCAPTCHAを無効にして既存機能を確認する運用が安全です。
- 本番更新前にはバックアップを取ってください。
- 本番更新後は、管理画面、無料PDF送信、CSV出力、購読者リスト、配信停止URL、reCAPTCHAを軽く確認してください。

## 7. zip作成・配布時の注意

- zipには `includes/` フォルダを必ず含めてください。
- zipには `docs/` フォルダを含めてください。
- zipには `README.md` / `README-ja.md` を含めてください。
- zipには `.git` を含めないでください。
- zipには `.codex` を含めないでください。
- `includes/` が欠けると、PHPエラーやプラグイン停止の原因になります。

## 8. 販売・納品時に説明すべきこと

- メール到達は保証できません。
- 到達率改善のため、SMTP設定が別途必要になる場合があります。
- SPF / DKIM / DMARCなどのDNS設定が関係する場合があります。
- 導入先のWordPress環境、サーバー環境、テーマ、他プラグインとの組み合わせにより、調整が必要になる場合があります。
- WordPress本体や他プラグインとの相性確認は、導入先環境ごとに必要です。
- 個人情報の取り扱い、メール配信ルール、同意管理の運用方針は、運用者側でも確認が必要です。
- 本プラグインは、専用のメール配信サービスやCRMを置き換えるものではありません。
