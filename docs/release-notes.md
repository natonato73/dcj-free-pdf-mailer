# DCJ Free PDF Mailer リリースノート

## 1. リリース概要

このドキュメントは、DCJ Free PDF Mailerの販売・納品・保守に向けた一区切りとして、現時点の機能、確認済み内容、運用上の注意、今後の改善候補を整理するものです。

本版は、DCJ Free PDF Mailerの第2実用版として位置づけます。無料PDF配布フォームの作成、PDFリンク付きメール送信、送信ログ管理、お知らせ受信同意、購読者リスト管理までを一通り扱える状態です。

販売前・納品前の確認資料として読めるようにまとめていますが、導入先のWordPress環境やメール送信環境によって追加確認が必要になる場合があります。

## 最新リリース項目

- 配信停止URL機能を追加
- 購読者一覧の検索・絞り込みを追加

- 管理画面からSite Key / Secret Key / Score Thresholdを設定可能
- reCAPTCHA無効時は従来どおり送信可能
- 購読者の個別削除機能を追加
- 削除前の確認ダイアログを追加
- 削除時も送信ログは保持

- 送信ログCSVと購読者リストCSVの運用を整理
- 「同意ありメールをCSV出力」ボタンを通常画面から非表示化
- 配信用メールアドレスは購読者リストの「購読中」CSVへ統一
- `{{newsletter_unsubscribe_block}}` を追加
- お知らせ受信に同意した場合だけ配信停止案内をメール本文に表示可能
- 配信停止URL処理を `includes/class-dcj-fpm-unsubscribe.php` に分離
- メール本文の運用性とコード保守性を改善
- 管理画面の運用性を改善

## 5. 運用上の注意

- PDF URLには、ブラウザで直接開けるURLを設定してください。
- `/wp-content/uploads/dlm_uploads/` 配下など、直接アクセスが制限されるURLは避けてください。
- メール到達は、サーバー環境、DNS設定、SMTP設定、受信者側のメール環境に依存します。
- お知らせ、販売案内、クーポン案内などに使う場合は、購読中のメールアドレスだけを使ってください。
- 配信用メールアドレスは、購読者リストでステータスを「購読中」に絞り込んでCSV出力してください。
- 送信ログCSVはフォーム送信履歴や証跡の確認用です。配信用メールアドレス一覧としては使わないでください。
- 配信停止の人には、お知らせや販売案内を送らないでください。
- 配信停止URLは、メール本文に `{{unsubscribe_url}}` を入れた場合にURL単体として送信されます。
- 通常は、お知らせ受信に同意した場合だけ案内を出せる `{{newsletter_unsubscribe_block}}` の利用がおすすめです。既存PDF設定のメール本文には自動追加されないため、使いたい場合は手動で追加してください。
- 配信停止URLをクリックした人は購読者リスト上で配信停止になります。無料PDFの受け取りには影響しません。
- reCAPTCHA v3を使うには、Google reCAPTCHA管理画面でv3のSite KeyとSecret Keyを取得し、実際にフォームを設置するドメインを登録してください。
- LocalWPなど登録ドメインと異なる環境では、reCAPTCHA検証が正常に通らない場合があります。ローカルではreCAPTCHA無効状態で既存機能を確認し、本番で有効化テストを行う運用が安全です。
- Score Thresholdは通常0.5から開始し、正常な送信が失敗する場合は0.3などに下げて確認してください。
- 購読者を削除する場合は、必要に応じて事前に購読者CSVを出力してください。削除されるのは購読者リストのみで、送信ログは保持されます。
- CSVファイルやメールアドレスは個人情報を含む可能性があるため、慎重に管理してください。
- 送信ログCSVと購読者リストCSVには、管理画面の検索・絞り込み条件が反映されます。
- 本番更新前には、WordPress本体、データベース、プラグイン、テーマ、アップロードファイルのバックアップを取ってください。
- 保守性改善として、管理画面メッセージ表示、CSV出力補助処理、reCAPTCHA処理に続き、配信停止URL処理も `includes/class-dcj-fpm-unsubscribe.php` へ分離しました。動作仕様は維持したまま、今後の保守をしやすくしています。

## 6. 販売・納品時に説明すべきこと

- メール到達は保証できません。
- 到達率改善のため、SMTP設定が別途必要になる場合があります。
- 導入先のWordPress環境、サーバー環境、テーマ、他プラグインとの組み合わせにより、調整が必要になる場合があります。
- WordPress本体や他プラグインとの相性確認は、導入先環境ごとに必要です。
- 個人情報の取り扱い、メール配信ルール、同意管理の運用方針は、運用者側でも確認が必要です。
- 本プラグインは、専用のメール配信サービスやCRMを置き換えるものではありません。
