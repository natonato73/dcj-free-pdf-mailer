# DCJ Free PDF Mailer v1.4.0 メルマガ送信MVPメモ

## 1. 目的

無料PDF配布フォームで集めた購読者に対して、WordPress管理画面から簡単なメルマガを手動送信できるようにする。

## 2. 実装範囲

- 管理画面のメルマガ送信セクション
- 送信対象の選択
  - 全員
  - 日本語購読者
  - 英語購読者
- 件名入力
- 本文入力
- テスト送信先メールアドレス入力
- テスト送信
- 送信前プレビュー
- プレビュー後の送信実行
- 送信結果表示
- 簡易送信ログ保存

## 3. 対象外

- 予約配信
- HTMLメールエディタ
- 画像添付
- 自動配信
- 外部メール配信サービス連携
- DBテーブル追加

## 4. 送信対象

送信対象は、配信停止ではない購読者のみとする。

既存データでは購読中のステータスを `subscribed` として保存しているため、v1.4.0では `active` と `subscribed` を送信可能な購読者として扱う。

- 全員: 送信可能な購読者すべて
- 日本語購読者: 送信可能かつ `lang = ja`
- 英語購読者: 送信可能かつ `lang = en`

## 5. 送信ログ

送信ログはWordPress optionに簡易ログとして保存する。

保存内容:

- 送信日時
- 件名
- 送信対象
- 対象件数
- 成功件数
- 失敗件数

DBテーブル追加は行わない。

## 6. 確認項目

- メルマガ送信セクションが管理画面に表示される
- 件名・本文未入力時に送信できない
- テスト送信で購読者DBが変わらない
- プレビューで件名、本文、送信対象、対象購読者数を確認できる
- プレビュー前に本送信できない
- 送信対象が全員、日本語、英語で切り替わる
- 本送信後に成功件数・失敗件数が表示される
- 本送信後に簡易ログが保存される
- 既存のPDF配布、購読者管理、CSV機能に影響がない
