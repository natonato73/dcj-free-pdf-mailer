# DCJ Free PDF Mailer v1.3.0 本番サイト適用手順書

## 1. 目的

- この手順書は、LocalWPで確認済みの DCJ Free PDF Mailer v1.3.0 を本番WordPressサイトへ安全に適用するためのものです。
- いきなり本番サイトへ適用せず、必ずバックアップと事前確認を行ってから進めます。
- 対象環境は WordPress / AFFINGER6 / ConoHa WING を想定しています。

## 2. 適用前の前提

- v1.3.0 はLocalWPで確認済みです。
- PHP構文チェックOKです。
- GitHub master反映済みです。
- 納品用zipは以下を使用します。

```text
/mnt/d/Dev_Projects/dcj-free-pdf-mailer/dcj-free-pdf-mailer.zip
```

- 本番適用前に必ずバックアップします。
- 本番サイトでのメール送信はサーバー環境に依存する可能性があります。
- CSVインポートはいきなり大量データを入れず、少数データで試します。

## 4. 本番バックアップ手順

### 4-1. WordPress管理画面でできるバックアップ

- DCJ Free PDF Mailerの購読者管理画面を開き、購読者CSVエクスポートを実行します。
- エクスポートしたCSVは、適用前のバックアップとして保存します。
- PDF設定一覧や編集画面を開き、現在の設定内容をスクリーンショットで保存します。
- 特にPDF名、PDFリンク、言語設定、表示内容が分かるように保存しておくと、戻し作業や確認がしやすくなります。

### 4-2. ConoHa WING側でのバックアップ

- ConoHa WINGの管理画面で、サイトデータのバックアップを取得します。
- データベースのバックアップも取得します。
- バックアップを取っただけで終わらず、復元できる状態か確認します。
- 自動バックアップがある場合でも、必要な時点のバックアップが残っているかを確認してから作業します。

### 4-3. プラグインフォルダの個別バックアップ

- FTPやConoHa WINGのファイルマネージャーを使い、以下のフォルダを保存しておきます。

```text
wp-content/plugins/dcj-free-pdf-mailer
```

- 保存先では、後から分かるようにフォルダ名に日付を付けます。

```text
dcj-free-pdf-mailer-backup-2026-05-03
```

- このバックアップは、更新後に問題が起きた場合に旧バージョンへ戻すために使います。

## 5. 本番適用手順

1. WordPress管理画面へログインします。
2. 左メニューから「プラグイン」>「新規追加」を開きます。
3. 「プラグインのアップロード」をクリックします。
4. `dcj-free-pdf-mailer.zip` を選択します。
5. すでにインストール済みの場合は「アップロードしたもので現在のものを置き換える」を選択します。
6. プラグインを有効化します。
7. プラグイン一覧で Version 1.3.0 と表示されていることを確認します。
8. DCJ Free PDF Mailerの管理画面を開きます。

注意:

- 可能ならアクセスの少ない時間に実施します。
- 更新前にバックアップが完了していることを再確認します。
- エラーが出た場合は無理に続けず、画面内容を保存して作業を止めます。

## 7. 適用後の動作確認

### 7-1. 確認用フォームのテスト

- 管理者メールアドレスでテストします。
- メールが届くか確認します。
- メール内のPDFリンクが正しいか確認します。
- メールが届かない場合は、迷惑メールフォルダに入っていないか確認します。

### 7-2. 実際の固定ページフォームのテスト

- 日本語フォームから送信します。
- 英語フォームから送信します。
- 送信後メッセージが正しく表示されるか確認します。
- メール本文が想定どおりか確認します。
- PDFリンクが正しいか確認します。

### 7-3. 購読者手動追加テスト

- `test-production-ja@example.com` を追加します。
- `test-production-en@example.com` を追加します。
- ja / en の表示が正しいか確認します。
- 同意確認チェックありで登録できることを確認します。

### 7-4. CSVインポート小規模テスト

- いきなり大量CSVを入れないようにします。
- 2〜3件の小さなCSVでプレビューします。
- プレビュー結果を確認してからインポートを実行します。
- `lang` は `ja` / `en` を指定します。
- `consent_confirmed` は `yes` / `1` / `true` を指定します。
- 配信停止済みのアドレスは戻さないようにします。

### 7-5. CSVエクスポート確認

- 適用後にCSVエクスポートできるか確認します。
- エクスポートしたCSVに、追加したテスト購読者や言語情報が含まれているか確認します。

## 8. 問題が出た場合の戻し方

- すぐに追加作業を止めます。
- エラー画面をスクリーンショット保存します。
- WordPress管理画面に入れる場合は、プラグインを無効化します。
- WordPress管理画面に入れない場合は、FTPまたはファイルマネージャーでプラグインフォルダ名を変更します。

```text
wp-content/plugins/dcj-free-pdf-mailer
wp-content/plugins/dcj-free-pdf-mailer-disabled
```

- 事前バックアップした旧プラグインフォルダに戻します。
- サイト全体に影響がある場合は、ConoHaのバックアップから復元を検討します。
- DB復元はサイト全体のデータに影響する可能性があるため、慎重に行います。
- 復元後は、管理画面、PDF設定、購読者一覧、メール送信を再確認します。
