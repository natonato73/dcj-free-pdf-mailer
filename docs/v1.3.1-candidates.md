# DCJ Free PDF Mailer v1.3.1以降の候補メモ

## 概要

v1.3.0 の開発完了後、実運用・本番適用・販売準備を進めながら検討する小改善候補を整理する。

## 今回実施したもの

- CSVテンプレート追加
- CSVインポート画面の説明強化
- 管理画面メッセージ微調整
- 既存購読者のlang未設定チェック

## 今回保留したもの

- 操作マニュアルPDF化
- 販売準備用の資料更新
- zip作成
- GitHub Release作成

## 候補

- CSVテンプレート追加
- CSVインポート画面の説明強化
- 管理画面メッセージ微調整
- 既存購読者のlang未設定チェック
- 操作マニュアルPDF化
- 本番サイト適用手順の整備

## 優先順位表

| 優先度 | 候補 | 目的 | メモ |
| --- | --- | --- | --- |
| 高 | 本番サイト適用手順の整備 | 本番反映時の作業漏れを防ぐ | v1.3.0で対応済み |
| 高 | CSVテンプレート追加 | CSV作成時の入力ミスを減らす | v1.3.1で対応済み |
| 中 | CSVインポート画面の説明強化 | 管理者が迷わず取り込めるようにする | v1.3.1で対応済み |
| 中 | 既存購読者のlang未設定チェック | 既存データの移行確認をしやすくする | v1.3.1で注意表示のみ対応済み |
| 中 | 操作マニュアルPDF化 | 納品・販売時の説明資料を整える | 今回は保留 |
| 低 | 管理画面メッセージ微調整 | 文言のわかりやすさを高める | v1.3.1で一部対応済み |

## 注意点

- DB構造変更は慎重に扱う
- 大量メール配信機能は対象外
- CRMやステップメール機能は対象外

## 最終方針

v1.3.1では、CSVインポート周りと管理画面表示の小改善のみを実施する。操作マニュアルPDF化や販売準備用の資料更新は今回は保留とし、必要に応じて次回以降に検討する。
