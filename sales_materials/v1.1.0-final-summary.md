# DCJ Free PDF Mailer v1.1.0 最終整理メモ

## 現在の最新版

- バージョン：v1.1.0
- 主な追加機能：メール送信診断パネル
- GitHubタグ：v1.1.0
- GitHub Release：DCJ Free PDF Mailer v1.1.0

## v1.1.0で追加した機能

- 管理画面にメール送信診断セクションを追加
- サイトURLを表示
- サイトドメインを表示
- WordPress管理者メールアドレスを表示
- 送信者名を表示
- 送信元メールアドレスを表示
- 送信元メールアドレスのドメインを表示
- 送信元メールアドレス形式を簡易チェック
- サイトドメインと送信元メールドメインを簡易比較
- 管理画面からテストメール送信
- SMTP / SPF / DKIM / DMARC確認を促す注意文を表示

## 注意点

メール送信診断は、メール到達トラブルの初期確認用です。

この機能は、メール到達を保証するものではありません。

メール到達率は、サーバー環境、DNS設定、SMTP設定、受信側メール環境に影響されます。

## 次回開発時の注意

次回開発を始める場合は、masterを直接触らず、作業ブランチを作る。

例：

git checkout -b feat/next-feature-name

作業後は、LocalWP確認、本番確認、PR作成、masterマージ、タグ作成、Release作成の流れで進める。
