## 主な機能

- 無料PDF申込フォームの表示
- PDFダウンロードリンク付きメール送信
- 日本語 / 英語のPDF設定
- PDF設定の新規登録・編集・削除・複製
- 送信ログ保存
- 送信ログ検索・絞り込み
- CSV出力
- 購読者リスト管理
- 購読中 / 配信停止のステータス管理
- 購読者の個別削除
- 配信停止URL
- 同意ありの場合だけ配信停止案内を表示する `{{newsletter_unsubscribe_block}}`
- メール送信診断
- サイトドメインと送信元メールドメインの簡易チェック
- テストメール送信
- SMTP / SPF / DKIM / DMARC 確認の注意表示
- reCAPTCHA v3対応

## インストール方法

WordPress管理画面で、以下の手順でインストールしてください。

1. **プラグイン**
2. **新規追加**
3. **プラグインのアップロード**
4. 納品zipファイルを選択
5. **今すぐインストール**
6. 既存プラグインがある場合は置き換え
7. プラグインが有効になっていることを確認

## 更新後の確認項目

更新後は、以下を軽く確認してください。

- 管理画面が開くこと
- 無料PDFフォームが表示されること
- PDFリンク付きメールが届くこと
- 送信ログが保存されること
- 購読者リストに反映されること
- CSV出力ができること
- 配信停止URLが動作すること
- メール送信診断パネルが表示されること
- テストメール送信を確認すること
- 送信元メールアドレスとサイトドメインの注意表示を確認すること
- reCAPTCHAを使う場合は本番ドメインで動作すること

## 注意事項

メール到達率は、サーバー環境、SMTP設定、DNS設定、受信者側のメール環境に影響されます。必要に応じて、SMTP設定やSPF / DKIM / DMARCの確認を行ってください。

メール送信診断は簡易チェックです。メール到達を保証するものではありません。必要に応じて、SMTP設定やSPF / DKIM / DMARCの確認を行ってください。

CSVにはメールアドレスが含まれるため、個人情報として慎重に管理してください。

お知らせ配信・販売案内・クーポン案内に使う場合は、購読者リストで **購読中** のメールアドレスだけを使用し、**配信停止** の方には送らないようにしてください。
