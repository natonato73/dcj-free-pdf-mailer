## 商品説明文

### サービス概要

WordPressサイトで、無料PDFをメール登録者へ自動送信できる仕組みを制作します。

「無料サンプルPDFを配布したい」  
「メールアドレスを取得して、今後のお知らせや販売案内につなげたい」  
「Contact Form 7などでは細かい運用が合わない」  

このような方向けに、専用のWordPressプラグインとして構築します。

無料PDFフォーム、PDFリンク付きメール送信、送信ログ、購読者リスト、CSV出力、配信停止URL、メール送信診断、reCAPTCHA v3など、実運用に必要な基本機能をまとめて実装します。

## 注意事項

メール到達率は、サーバー環境、SMTP設定、DNS設定、受信者側のメール環境に影響されます。必要に応じて、SMTP設定やSPF / DKIM / DMARCの確認が必要です。

メール送信診断は簡易チェックです。メール到達を保証するものではありません。必要に応じてSMTP設定やSPF / DKIM / DMARCの確認が必要です。

また、メールアドレスを扱うため、CSVファイルや購読者リストは個人情報として慎重に管理してください。

本サービスは、専用メール配信サービスやCRMを完全に置き換えるものではありません。小規模な無料PDF配布、見込み客リスト管理、初期導線作成に向いた仕組みです。

## v1.2.1 追記：管理用CSVとメール配信用CSVの整理に対応

DCJ Free PDF Mailer v1.2.1では、購読者リスト周辺のCSV出力ボタン名を整理し、管理・バックアップ用CSVとメール配信用CSVの違いを分かりやすくしました。

この機能により、購読者リストの保管・確認用CSVと、メール配信サービスへ手動インポートするためのCSVを分けて運用しやすくなっています。

主な内容：

- **管理・バックアップ用CSV出力** に対応
- **メール配信用CSV出力（全言語）** に対応
- **日本語メール配信用CSV出力** に対応
- **英語メール配信用CSV出力** に対応
- メール配信用CSVは購読中のみを対象
- 配信停止者はメール配信用CSVから除外
- メール配信用CSVのファイル名に `all` / `ja` / `en` を付与
- 既存の送信ログCSVは従来どおり維持

メール配信用CSVの列：

- メールアドレス
- 言語
- 登録元PDF ID
- 登録元タイトル
- 最終同意日時

注意：

メール配信用CSVは、メール配信サービスへの手動インポートを補助するためのCSVです。  
メール配信前には、CSV内容、同意状況、配信停止者の扱い、配信サービス側の同意管理・配信停止管理を必ず確認してください。
