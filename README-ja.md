# DCJ Free PDF Mailer

DCJ Free PDF Mailerは、WordPress上で無料PDF配布フォームを作成するためのプラグインです。

ユーザーがフォームにメールアドレスを入力すると、設定したPDFリンクをメールで送信できます。複数のPDF設定を管理でき、日本語・英語の配布導線にも対応しやすい構成になっています。

## 概要

このプラグインでは、WordPress管理画面からPDF配布用の設定を作成し、ショートコードを使って固定ページや投稿にフォームを表示できます。

主な流れは次の通りです。

1. PDF設定を作成する
2. ショートコードをページに貼る
3. ユーザーがメールアドレスを入力する
4. PDFリンク付きメールが送信される
5. 管理画面で送信ログを確認する

PDF設定はWordPressのoptionに保存されます。専用のDBテーブルは使用していません。

メールアドレスを受け取り、PDFリンクを案内するシンプルな配布導線に向いています。

## ショートコード例

```text
[dcj_free_pdf id="dcj-001-ja"]
```

`id` には、管理画面で作成したPDF設定の管理IDを指定します。

## 基本的な使い方

1. 配布したいPDFをWordPressのメディアライブラリにアップロードします。
2. WordPress管理画面で **DCJ Free PDF** を開きます。
3. PDF設定を新規作成します。
4. PDF URLを設定します。
5. メール件名・メール本文を確認します。
6. ショートコードを固定ページや投稿に貼ります。
7. テスト送信します。
8. 送信ログを確認します。

初めて使う場合は、公開前に自分宛てのテスト送信を行うことをおすすめします。

## PDF URLの注意

PDF URLには、ブラウザで直接開けるURLを指定してください。

避けた方がよい例:

```text
/wp-content/uploads/dlm_uploads/
```

上記のような直接アクセスが制限される場所のURLでは、メール内のPDFリンクが `Forbidden` になる場合があります。

PDFリンクが開けない場合は、通常のWordPressメディアライブラリにPDFをアップロードし、そのURLを使ってください。

## メール到達に関する注意

このプラグインはWordPressの `wp_mail()` を使ってメール送信します。

メールの到達状況は、サーバー環境、DNS設定、SMTP設定、受信者側のメール環境などに影響されます。

起こりうること:

- メールが迷惑メールフォルダに入る
- Gmailなどでサーバー経由表示になる
- 受信者側の設定によりメールが見つかりにくい

到達率を改善したい場合は、WP Mail SMTPやFluentSMTPなどのSMTP設定を検討してください。SPF / DKIM / DMARCなどのDNS設定が関係する場合もあります。

## インストール方法

1. プラグインzipを用意します。
2. WordPress管理画面を開きます。
3. **プラグイン** > **新規追加** を開きます。
4. **プラグインのアップロード** を選択します。
5. zipファイルを選択してインストールします。
6. プラグインを有効化します。
7. **DCJ Free PDF** 管理画面を開きます。
8. PDF設定を作成します。

## 注意事項

- メール到達は保証できません。
- PDF URLは必ず事前に開いて確認してください。
- PDF送付とお知らせ配信は分けて考えてください。
- お知らせや販売案内を送る場合は、同意ありのメールアドレスだけを使ってください。
- お知らせや販売案内を送る場合は、購読中のメールアドレスだけを使ってください。
- 配信停止の人には送らないでください。
- 配信停止URLは、メール本文に `{{unsubscribe_url}}` を入れた場合だけ送信されます。
- 配信停止URLをクリックした人は購読者リスト上で配信停止になりますが、無料PDFの受け取りには影響しません。
- 送信ログ・購読者リストの検索は、管理画面に表示されている範囲内での絞り込みです。
- メールアドレスやCSVは個人情報として慎重に管理してください。
- 購読者CSVやメールアドレスは慎重に管理してください。
- 本番導入前にはバックアップを取ってください。
- SMTP情報、パスワード、APIキーなどはGitHubやドキュメントに保存しないでください。
- 他のテーマやプラグインとの組み合わせにより、調整が必要になる場合があります。
